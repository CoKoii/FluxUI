# 🚀 FluxUI - 快速开始

## 两个核心命令

### 1. 创建组件
```bash
pnpm new
```
输入组件名称，自动生成所有必要文件。

### 2. 发布
```bash
pnpm release
```
一键发布到 npm（自动处理构建、版本更新、Git 提交等）。

## 开发流程
```bash
pnpm new             # 创建组件
pnpm dev:core        # 开发调试
pnpm release         # 发布
```

详细说明见 [AUTOMATION_GUIDE.md](AUTOMATION_GUIDE.md)
